Lexical Structure

The lexical structure of a programming language is the rules that govern how we write source code to this language.

Tokens

A token is a elementary words or reserve words of a language.

Identifiers

A identifier is a word that references a variable, upper and lower letters can be used and the underscore at the beginning and after we may add numbers and more letters. Don't use Homoglyphs.

Python identifier convention

  • Classes: start with uppercase

  • Private: start with underscore

  • Strongly private: start with two underscore

  • Variables: Snake case

Reserved Words

Python has 35 reserved words that we can't used how identifiers. There is another type that is soft words this means that the words are reserved if has a certain structure.

 
#this code print the all python keywords
import keyword
#Reserved words
print(keyword.kwlist)  
#Soft words
print(keyword.softwlist)

Escape Sequence

A escape sequence is a way to include any character inside a string literal.

  • \Ignore the end of a line.
  • \\Backslash
  • \'Single quote
  • \"Double quote
  • \aBell
  • \bBackspace
  • \fForm feed
  • \nNewline
  • \rCarriage return
  • \tTab
  • \vVertical tab
  • \ DDDOctal value
  • \ XXHexadecimal value
  • \NUnicode character
  • \oAny other character
  • \uUnicode character

There is a handy notation to use a multiples backslash in the same literal. We add r or R at the start of the quote.


# this is a backslash with following a character 
not_escape_sequence = r"\n"
print(not_escape_sequence)

bytes

A bytes is an ordered list of int with the size of 1 byte. We can use the prefixed with b in string literal.


# Create a bytes representation of your name
print(rb"Hello")
# Decode
print(rb"Decoding the same message".decode())
